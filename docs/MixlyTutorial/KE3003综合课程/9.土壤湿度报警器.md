# 3.6.9 土壤湿度报警器

## 3.6.9.1 简介

当土壤湿度传感器感应到花盆的土壤湿度低于设置的阈值时，语音模块就会发出警告提示音“请注意花盆土壤湿度过低，请尽快浇水”

## 3.6.9.2 控制指令表

**消息号表：**

| 消息号 |              播报语音              |
| :----: | :--------------------------------: |
|   17   | 请注意花盆土壤湿度过低，请尽快浇水 |
